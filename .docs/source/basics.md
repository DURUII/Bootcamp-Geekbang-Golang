## 变量声明

Go 静态语言编译器必须要程序员声明内存的边界信息（变量类型）。
Go 语言支持一行声明多个变量，支持变量声明块，支持省略类型信息的声明、`:=`?/短变量声明。

### 变量零值

| 原生类型 | 默认值 |
|::|:---:|
| 整数 | 0|
| 浮点数 | 0.0 |
| 布尔 | false|
| 字符串 | ""  |
| 指针、接口、切片、`channel`、`map`和函数 | `nil` |

而数组、结构体是复合类型是他们组成元素都为零值的结果。
